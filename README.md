# 🌐 Web Design Learning

Repository này được tạo ra trong quá trình mình **học môn Thiết Kế Web**.
Mục tiêu là lưu lại các bài thực hành, ví dụ và project nhỏ trong quá trình học **HTML, CSS và JavaScript**.

Thông qua repository này, mình luyện tập cách:

* Xây dựng giao diện website
* Thiết kế bố cục trang web
* Tạo hiệu ứng tương tác bằng JavaScript
* Áp dụng kiến thức vào các project thực tế

---

# 📚 Nội dung đã học

Trong quá trình học thiết kế web, mình đã thực hành các nội dung sau:

### 1️⃣ HTML cơ bản

* Cấu trúc trang web
* Các thẻ HTML phổ biến
* Tạo form nhập dữ liệu
* Nhúng hình ảnh, video và âm thanh

Ví dụ:

```html
<h1>Xin chào</h1>
<p>Đây là trang web đầu tiên của tôi.</p>
```

---

### 2️⃣ CSS thiết kế giao diện

* Thiết kế layout
* Sử dụng Flexbox
* Responsive cho nhiều màn hình
* Hiệu ứng hover và animation

Ví dụ:

```css
.container{
    display: flex;
    justify-content: center;
    align-items: center;
}
```

---

### 3️⃣ JavaScript cơ bản

* Xử lý sự kiện
* Thay đổi nội dung HTML
* Tạo hiệu ứng động

Ví dụ:

```javascript
function showMessage(){
    alert("Xin chào!");
}
```

# ▶️ Cách chạy project

Các project trong repository là **website tĩnh**, nên có thể chạy rất đơn giản.

### Cách 1

Mở file:

```
index.html
```

bằng trình duyệt:

* Google Chrome
* Microsoft Edge
* Firefox

---

### Cách 2 (khuyến khích)

Sử dụng **Live Server trong VS Code**

1. Cài extension **Live Server**
2. Chuột phải vào `index.html`
3. Chọn **Open with Live Server**
